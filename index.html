Library Management System

I developed a comprehensive Library Management System using Java and MySQL to streamline core library operations. The system includes key features such as book management (add, update, delete), borrowing and returning with real-time tracking of due dates, and automatic fine calculation for overdue books. It uses MySQL as a secure backend to store and manage data efficiently. Additionally, the system integrates SMTP-based email alerts to notify users about due dates and other important updates, ensuring better user engagement and accountability.

Java (Swing,JDBC) MySql SMTP Server
